4). Compiler VS Interpreter (Most Important)
-----------------------------------------------------------------
✅ Compiler
A Compiler translate the entire code at one into machine code.

10-Lines NewFile - nameof the file.exe-Output

Example:-
------------------------
* C 
* C++
* Java (Java Used both Compiler + Interpreter concept)
------------------------------------------
Characteristics:
-------------------
* Faster Execution
* Errors shown after full compilation.
------------------------------------------------------------------------------------

✅ Interpreter

An interpreter translates the program line by line.
--------------------------------------------------------------------
Example:-
--------------------------------
* Python
* JavaScript
* Ruby
--------------------------------------------------------------------
Characteristics:
------------------------
* Easier Debugging
* Slower than compiled language
* Executes directly
------------------------------------------------------------------------------
Example: If error is in line 10, Python executes line 1-9 and stop at line 10.
-----------------------------------------------------------------------------------

Comparison Table
--------------------------------------------------------
|    Feature	|   Compiler	|  Interpreter	|
--------------------------------------------------------
|Translation	|Whole Program	| Line by line	|
----------------------------------------------------
|Ex-Speed	|Fast		|Slower		|
--------------------------------------------------------
|Error-Display	|After Compilation | Immediately|
------------------------------------------------------------
|Example	|C,C++		| Python, JavaScript|
-------------------------------------------------------------------------------
