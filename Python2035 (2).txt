1). What is Python?
--------------------------------------
>> Python is high-level, general-purpose, interpreted programming language that is designed to be easy to read, easy to write, and easy to learn.
---------------------------------------------------------------------------------------

>> Python is a language that helps us to communicate with computer using simple English-like syntax.
It is widely used because it reduces development time and works in almost every domain.
---------------------------------------------------------------------------------------------
Example-

print("Hello world")
-----------------------------------------------------------------------------------------------
2). Who Created Python?
----------------------------------------
>> Python was created by Guido Van Rossum in 1989 and official released in 1991.
-----------------------------------------------------------------------------------------

Python's name is inspired by the comedy show "Monty Python's Flying Circus", not the snake.

3). High-Level vs Low-Level language
------------------------------------------------------------------------------------------
✅ High-Level Language
------------------------------------------
A high-level language is closer to human language and easier to understand.
-------------------------------------------------------------------------------------------
Example:- 
-------------------
>> Python 
>> Java 
>> C#
>> JavaScript
>> etc.
-----------------------------------------------------------------------------------------------------

Benefits:
----------------
>> Easy Syntax
>> Faster Development
>> Easy Debugging --> Bug --> Debug
----------------------------------------------------------------------------------------------

✅ Low-Level Language
------------------------------
Low-Level Languages are close to machine and difficult for humans.
-------------------------------------------------------------------------------------------
Example:-
-----------------
>> Assembly Language
>> Machine Code(0,1)
--------------------------------------------
>> Benefits
-------------
* Very Fast execution
* Direct hardware control
-----------------------------------------------------------

Notes:

High-level languages are like English, Low-levelnguages are like binary signals.
----------------------------------------------------------------------------------------------

4). Compiler VS Interpreted
------------------------------------------
A compiler translate the entire code at once into machine code.
---------------------------------------------------------------------------------------------
Example:

* C
* C++
* Java(Java uses both compiler+ Interpreter concept)

----------------------------------------------------------------------------
>> Characteristics:
---------------------
>> Faster Execution
>> Errors shown after full compilation
----------------------------------------------------------------------------------------

>> ✅ Interpreter

An interpreter translate the program line by line.

Examples:

* Python
* JavaScript
* Ruby

---------------------------------------------------------------------------

>> Characteristics:
-------------------------------
>> Easier Debugging
>> Slower than compiled languages
>> Executes directly.
-------------------------------------------------------------------------------

✅ Python Installation(Windows/Mac/Linux)
------------------------------------------------

1). Install Python on Windows
--------------------------------------
step-1: Download Python
------------------------
1. Open Browser
2. Search Python official website
3. Open: python.org
4. Clock: Downloads
5. Download latest version(Example: Python 3.14.2)

------------------------------------------------------------------------------
Step - 2: Run Setup file
-----------------------
1. Open download .exe file
2. Important:  Tick the checkbox ✅Add Python to PATH
3. Click Install Now

Not:
if you forget to tick "Add Python to PATH", Python will not run in CMD properly.
----------------------------------------------------------------------------------------------------

Stp-3: Verify Installion
--------------------------
1. Open Command prompt(CMD)
2. Type:
	python --version
---------------------------------------------------------
Bash-
	python -v
--------------------------------------------------

Step:-4: Check pip installation
-------------------------------------------------
pip[preferred installer program] is used to install libraries.
--------------------
pip --version
----------------------------------------------------------------------------------------------

>> 2). Install Python on Mac
----------------------------
Step 1: Download Python
	1. Go to python.org
	2. Download Mac version.pkg
--------------
Step 2:
	1. Open the downloaded file
	2. Click Continue -> Install
-----------------
Step 3:
	Open terminal and type
	python3 --version
Bash
	pip3 --version
